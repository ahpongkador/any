<h2>What is a python list</h2>
<h4> -A Python list is a collection of items which is ordered and changeable. Lists allow duplicate members. They are one of the most versatile data types in Python.</h4>
  
<h2>How to access, change, add, and remove items in a list?</h2>
  <h4>Access: List items are indexed and you can access them by referring to the index number</h4>
  <h4>Change: To change the value of a specific item, refer to the index number</h4>
  <h4>Add: To add an item to the end of the list, use the append() method</h4>
  <h4>Remove: The remove() method removes the specified item</h4>
  <h2> How to sort a list?</h2>
  <h4>List objects have a sort() method that will sort the list alphanumerically, ascending, by default</h4>
  <h1>Selection Statements</h1>
  <h4>IF statement: An "if statement" is written by using the if keyword.</h4>\
  <h4>IF-else: The else keyword catches anything which isn't caught by the preceding conditions.</h4>
  <h4>IF-elif: The elif keyword is Python's way of saying "if the previous conditions were not true, then try this condition".</h4>
  <h2>String Methods</h2>
  <h4>string.lower(): The lower() method returns the string in lower case</h4>
  <h4>string.upper(): The upper() method returns the string in upper case</h4>
  <h4>string.title(): Converts the first character of each word to upper case</h4>
  <h4>string.replace(old, new): Returns a string where a specified value is replaced with a specified value</h4>
  <h1> Subprograms in python</h1>
  <h2>What is a subprogram?</h2>
  <h4>A subprogram is a small part of a program that does a specific job. In Python, this is called a function.</h4>
  <h2>how to create one?<h2>
    <h4>You make a subprogram (function) using the def keyword, give it a name, and write the code it should run.</h4>
    <h2>How to use it?</h2>
    <h4>You use a subprogram by calling its name and giving it any needed information (arguments). It will do its job and give back a result if needed.</h4>
